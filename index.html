담당자 부재에 따른 지능형 시스템 구축 방안

1. 현황 및 추진 배경

메인 담당자 공백에 따른 운영 위기

기존 방송 시스템은 1인의 숙련된 작업자가 PPT 제작, 카메라 제어, 음향 믹싱을 모두 수동으로 처리해 왔습니다. 현재 해당 인원의 공백이 확정되어, 예배 방송이 우려되는 실정입니다.

  • 인력 공백: 사람 자체가 없음(나도 안할거임 하지만 장로님이 신뢰해 주니까...)
  • 업무 비효율: 업무량이 많고 심적 부담으로 새로운 담당자도 오래 못함
  • 지속 가능성 결여: 봉사에 의존하는 구조로는 매번 동일한 위기 반복

2. 자동화 구축 상세 계획

첫 번째: 방송 준비 자동화 (PPT 제작)

주보 데이터 수집부터 최종 송출용 파일 생성까지의 과정을 코딩 기반으로 자동화합니다.


구글 폼 수집

구글 시트 입력

데이터 매핑

구글 슬라이드 / 한컴오피스
  • 구글 폼을 통해 예배 정보를 텍스트 데이터로 직접 수집
  • Apps Script를 활용하여 구글 슬라이드 템플릿에 실시간 데이터 주입
  • 한컴오피스(HWP) 기반 템플릿 연동 시스템 구축으로 기존 시스템 유지

두 번째: 방송 진행 자동화 (PPT, 카메라)

현장의 실시간 데이터(음성, 영상)를 분석하여 슬라이드 전환과 카메라 앵글을 자동으로 제어합니다.

STT (Speech-to-Text) 실시간 오디오를 텍스트로 변환하여 현재 진행 중인 멘트 파악
Object Detection 카메라 영상 내 객체 유무, 움직임, 위치를 감지하여 상황 인지
Embedding Model 비교 제어 수집된 데이터(STT, 영상)를 슬라이드에 사전 부여된 메타데이터와 임베딩 모델로 비교 분석하여 최적의 슬라이드 및 카메라 앵글로 자동 전환

음향 자동화 부분은 기술적 검토 및 현장 안정성 확보를 위해 현재 보류 중입니다.

세 번째: 기타 업무 자동화 (보류)

기타 업무 자동화는 우선순위에 따라 현재 보류 상태이며, 향후 시스템 안정화 단계에서 재검토할 예정입니다.

3. 추진 방향 및 기대 효과

  • 설계 목표: 누구나 AI로 유지보수가 가능해질 때까지 유지보수 최소화
  • 나의 역할: 시스템의 전체 설계와 초기 구축을 담당하며, 향후 기술적 문제 해결을 위한 기술 자문 역할 수행
  • 기대 효과: 최소한의 업무로 최소한의 인력 사용
